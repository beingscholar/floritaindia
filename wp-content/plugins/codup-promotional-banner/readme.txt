== Description ==

Promotional banner provides you to show banner related to discount offers, any alert or notifications etc on your site, this plugin works in slider or just one-line content.

*Features*

* This plugin provides admin to change the banner background and font color.
* This plugin provides the option to set the banner on all pages or specific pages.
* Promotional banner allows the admin to set button on banner through which customer navigating to target link.
* Promotional banner provides the delay time to move the slider.
* This plugin allow admin to set the expiry date after the expiry date banner will not show on frontend.
* This banner provides admin to set the banner on three different position but banner will show one position at a time, the positions are: 1. Top 2. Top Fixed 3. Bottom
* Admin can set the font size and height of the banner which will show on site.
* Admin can Enable/Disable the banner through which banner will show on frontend or not.


== Installation ==
1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the Plugins screen in WordPress
3. User dashboard -> promo banner -> general Setting
